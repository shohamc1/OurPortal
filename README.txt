OurPortal
=========
A module bidding solution for SUTD.

Master branch available at https://op-master.shohamc1.com.
Dev branch available at https://ourportal.shohamc1.com.

Technical Details
-----------------
- Frontend Framework: Gatsby
- CSS Framework: TailwindCSS

- Frontend Deployment: Vercel
- Seach Provider: Algolia
- Authentication Provider: Firebase Authentication
- Live Counter: Firebase Realtime Database

Backend code: https://github.com/xvnyv/ourportal_backend
